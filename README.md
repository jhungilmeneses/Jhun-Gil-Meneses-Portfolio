# Jhun-Gil-Meneses-Portfolio
First Repository in GitHub
A modern personal portfolio website showcasing my projects, skills, and professional background as a developer. This web application serves as my digital resume and project gallery where visitors can explore my work and contact me.

This portfolio web application is designed to present my work, skills, and experience in a clean and interactive way. It highlights my development projects, technical skills, and provides an easy way for potential employers, collaborators, or clients to learn more about me.

The website is deployed using Firebase Hosting, ensuring fast performance and reliable availability.

✨ Features

👤 About Me Section – Brief introduction and background

💻 Projects Showcase – Displays my development projects

🧠 Skills Section – Technologies and tools I use

📱 Responsive Design – Works on desktop, tablet, and mobile

⚡ Fast Hosting – Deployed with Firebase Hosting

📬 Contact Section – Allows visitors to connect with me
